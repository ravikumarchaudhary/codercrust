Vector

Last Updated: 12july 2021


Vector are just like arrays stored in the contigous memory locations but unlike arrays vectors uses dynamic memory allocation in which iterators are used for accessing and traversing through the data .
Internally, in a vector container always allocates some extra memory for the possible growth so whenever the capacity is measured it is always greater than normal .Therefore, compared to arrays, vectors consume more memory in exchange for the ability to manage storage and grow dynamically.Also as compared to other dynamic sequence containers like deques, lists and forward_lists , vectors are also efficient when it comes to add or remove elements from its end. However for operations that involve inserting or removing elements at positions other than the end, they perform worse than the others, and have less consistent iterators and references than lists and forward_lists.



Some member functions of vector class are:

    Iterators
  • begin() : Returns an iterator pointing the first element in vector
  • end() : Returns an iterator pointing the last element in vector
    •
  • rbegin() : Returns a reverse iterator pointing the last element in vector
    •
  • rend() : Returns a reverse iterator pointing the first element in vector
    •
  • cbegin() : Returns a constant iterator pointing the first element in vector
    •
  • cend() : Returns a constant iterator pointing the last element in vector
    •
  • crbegin() : Returns a constant reverse iterator pointing the last element in vector
    •
  • crend() : Returns a constant reverse iterator pointing the first element in vector
    •
    Modfiers
  • assign : It Assigns values to vector
    •
  • push_back : inserts the element from back of vector.
  • pop_back : delete element from end of vector.
    •
  • erase : erase elements
    •
  • insert : insert element at place.
    •
  • swap : Swap content of one vector with another.
    •
  • clear : clear the vector.
    •
  • emplace : construct new element and assign to vector.

    Capacity
  • size : Returns the size of vector
    •
  • max_size : Returns the maximum size
  • resize : Changes the size of vector
    •
  • empty : Checks whether the vector is empty or not
    •
  • capacity : Returns the akkocated memory of vector
    •
  • reserve : Requests for the change in the capacity.
    •

    Element access
  • operator[] : access element
    •
  • at : access element
  • front : access front element
    •
  • end : access end element

  • data : access data
