Que1. Which of the following statement is correct?
Ans: a. Class is an instance of object
        b. Object is an instance of a class
        c. Class is an instance of data type
        d. Object is an instance of data type

    • b. Object is an instance of a class



Que2. What is the difference between structure and class in C++?
Ans: a. A struct cannot have member functions.
        b. A class has a default privacy specification of public.
        c. A class has a default privacy specification of private
        d. A struct cannot have private members.

    • C. A class has a default privacy specification of private



Que3. If you want to write multiple functions in a class with same name, then what C++ feature will you use?
Ans: a. Function overriding
        b. Encapsulation
        c. Function overloading
        d. None

    • C. Function overloading



Que4. What is the difference between an object and a class?
Ans: a. An object is an extension of the class construct whose default access privilege is public.
        b. The term object is just another way of referring to the public data members of a class.
        c. An object is an initialized class variable.
        d. A class is an initialized object variable

    • c) An object is an initialized class variable.



Que5. If 2 classes derive one base class and redefine a function of base class, also overload some operators inside class body. Among these two things of function and operator overloading, where is polymorphism used?
Ans: a. Function overloading only
        b. Operator overloading only
        c. Both of these are using polymorphism
        d. Either function overloading or operator overloading because polymorphism can be applied only once in a program

    • d) Either function overloading or operator overloading because polymorphism can be applied only once in a program



Que6. Constructor is executed when _____.
Ans: a. An object goes out of scope.
        b. A class is declared
        c. An object is created
        d. An object is used

    • C) An object is created



Que7. Which of the following is a valid class declaration?

Que8. Which of the following statements is correct when a class is inherited publicly?
Ans: a. Public members of the base class become protected members of derived class.
        b. Public members of the base class become private members of derived class.
        c. Private members of the base class become protected members of derived class.
        d. Public members of the base class become public members of derived class.

    • d) Public members of the base class become public members of derived class.

Que10. A…………can only be called by another function that is a member of its class.

Que11. Static variables are associated with the class itself rather than with any class object, they are also known as……………

Que12. Static variables are like…………………as they are declared in a class declaration and defined in the source file.

Que13. Which statement is true for static data member of a class?
Ans: a. A static member function can access only static data member of a class and static data member is shared among all the objects of a class.
        b. A Static data member is shared among all the objects of a class.
        c. A Static data member can be accessed directly from main.
        d. A Static member function can access only static data member of a class.

Que14.
        class box
       {
          private:
            integer height, weight;
          public:
          function input(int a, int b)
          {
            height=a;
            weight=b;
          }
        };
        function main()
        {
            box b1,b2;
        }

Ans: a. “Box” is a class with b1 and b2 as its objects, with “height” and “weight” as its attributes.
        b. “Box” is a class with b1 and b2 as its objects, with “height” and “weight” as its objects.
        c. “Box” is a class with b1, b2, “height” and “weight” as its attributes.
        d. “Box” is a class with b1, b2, “height” and “weight” as its objects

    • a) “Box” is a class with b1 and b2 as its objects, with “height” and “weight” as its attributes



Que15. Which of the following statements is correct?
Ans: a. First time method of a class is called, the constructor method is called.
        b. Every time method of a class is called, the constructor method is called.
        c. Every time an instance of a class is created, the constructor method is called.
        d. none

Que17. A friend function declaration:
Ans: a. gives nonmember functions access to hidden class members
