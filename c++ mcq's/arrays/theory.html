Que1. What will be the resulting array after rotating arr[]={1,2,3,4,5} by 2?
Ans: a. {2,1,3,4,5}
        b. {3,4,5,1,2}
        c. {4,5,1,2,3}
        d. {1,2,3,5,4}

    • {3,4,5,1,2}


Que2. Let x be an array. Which of the following operation are illegal?
      I. ++x
      II. x + 1
      III. x++
      IV. x*2
Ans: a. I and II
        b. I,II and III
        c. II and III
        d. I,II and IV
        e. III and IV

    • d) I,III and IV


Que3. Array passed as an argument to a function is interpreted as:
Ans: a. Address of the array
        b. Values of the first element of the array
        c. Address of the first element of the array
        d. Number of element of the array

    • c) Address of the first element of the array


Que4. Let A be a square matrix of size n x n. Consider the following program. What is the expected output?
  using namespace std;
  #include< bits/stdc++.h>
  int main()
  {
    C = 100;
    for i = 1 to n do
          for j = 1 to n do
          {
                Temp = A[i][j] + C;
                A[i][j] = A[j][i];
                A[j][i] = Temp - C;
          }
    for i = 1 to n do
          for j = 1 to n do
                Output(A[i][j]);
  }
Ans: a. The matrix A itself
        b. Transpose of matrix A
        c. Adding 100 to the upper diagonal elements and subtracting 100 from diagonal elements of A.
        d. None of the above

    • a) The matrix A itself



Que5. Which of the following is the correct way to define the function fun() in the program below?
  #include< stdio.h>
  int main()
  {
    int a[3][4];
    fun(a);
  }
Ans: a. void fun(int p[][4]){}
        b. void fun(int *p[][4]){}
        c. void fun(int *p[3][4]);
        d. void fun(int *p[4]){}

    • a) void fun(int p[][4]){}


Que6. What does the following declaration mean?
  int (*ptr)[10];
Ans: a. ptr is array of pointers to 10 integers
        b. ptr is a pointer to an array of 10 integers
        c. ptr is an array of 10 integers
        d. ptr is an pointer to array

    • b) ptr is a pointer to an array of 10 integers
