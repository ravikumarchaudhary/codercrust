Que6. What is the output of the code?
  using namespace std;
  #include< bits/stdc++.h>
  class access
  {
      int a = 10;
    void disp()
    {
      cout<< "a: "<< a;  
    }
  };
  int main()
  {
    access a;
    a.disp();
    return 0;
  }
Ans: a. a: 10
        b. a: 20
        c. Runtime error
        d. Compile time error: Because disp() is private

    • d) Complile time error: Because disp() is private

    • Explanation: When the access specifiers are not specified then they are taken as private by default. The private members of a class can be accessed only with in the class and can not be accessed using . operator outside of a class.



Que7. What it the output of the code?
  using namespace std;
  #include< bits/stdc++.h>
  struct A
  {
      int a;
    A()
    {
      a = 23;
    }
  };
  struct B:A
  {
    B():A()
    {
      a = 45;
    }
  };
  int main()
  {
    A obj1;
    B obj2;
    cout<< obj1.a<< " "<< obj2.a<< " "<< obj2.A::a;
  }
Ans: a. 23 23 45
        b. 45 45 45
        c. 23 45 45
        d. compile time error

    • c) 23 45 45

    • No compile time error. Because in structure the default access specifier is public. When we inherit a structure from another structure the default mode of inheritance is public. So a of class A is remains public in struct B. So accessing data member of structure outside of the structure is legal. Obj1.a and obj2.a prints their respective copies a as 23 and 45. At last obj2.A::a prints base copy of a as part of the derived. So prints 45



Que8. What is the output of the code?
  using namespace std;
  #include< bits/stdc++.>
  class Test
  {
    private:
      int data;
    public:
    Test(){ data = 0;}
    int getData(){return data;}
  };
  int main()
  {
    Test t;
    int *ptr = (int*)&t;
    *ptr = 10;
    cout<< t.getData();
    return 0;
  }
Ans: a. 10
        b. 0
        c. Compile Time Error: Because typecasting the address of an object to int pointer is illegal
        d. Accessing private data using pointer is not allowed

    • a) 10

    • Accessing private data using pointer is possible. After typecasting address of member data is assigned to pointer ptr. In next line value at ptr is changed to 10. So, t.getData() prints the value of member data as 10.
