Que1. Which among the following is correct for the code given below?
  class A
  {
      int marks;
    private:
    student(){}
    public:
    student(int x)
    {
      marks = x;
    }
  };
  Ans: a. The object can never be created
        b. The object can be created without parameters
        c. Only the object with only 1 parameter can be created
        d. Only the object with some parameters can be created

  using namespace std;
  #include< bits/stdc++.h>
    class A

    private: int c;
    public: int a;
    public: int k;
    public:

      B obj;

Que6. Identify the correct statement?
  using namespace std;
 #include< bits/stdc++.h>
    class A
    {
      protected:
        int c;
      A()
      {
        c = 67;
     }
    };
    class B:private A
    {
      public:
        int c;
    B():A()
    {
        c = 90;
    }
  };
  int main()
  {
    B obj;
    cout<< obj.c;  
  }
Ans: a. It will raised a compile time error, because c is private in A
        b. It will raised a compile time error, because c is redefined in B
        c. No compile time error, because c is public in B. And accessing c from main is legal and display 67
        d. No compile time error, because c is public in B for redefinition. And accessing c from main is legal and display 90.

    • d) No compile time error, because c is public in B for redefinition. And accessing c from main is legal and display 90.



Que7. Identify the legal statements?
  using namespace std;
  #include< bits/stdc++.h>
  struct A
  {
      int a;
    A()
    {
      a = 23;
    }
  };
  struct B:A
  {
    B():A()
    {
      a = 45;
    }
  };
  int main()
  {
    A obj1;
    B obj2;
    cout<< obj1.a;
    cout<< obj2.a;
    cout<< obj2.A::a;
  }
Ans: a. only line1
        b. only line1 and line2
        c. only line2 and line3
        d. All are correct

    • a) only line1

    • Only line1 is a legal statement. Accessing a at line2 and line3 using the object of B is illegal, because A is inherited to B in private mode.
